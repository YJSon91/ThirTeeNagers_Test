## 게임 개요
#### 스파르타코딩클럽 내일배움캠프 - Unity 입문 팀 프로젝트

Unity를 활용한 2D 러닝 액션 장르 게임
스테이지를 클러어 하며 점수를 획득하는 게임

## 팀원 소개

| 이름   | 역할 | 담당 업무                                          |
|--------|------|----------------------------------------------------|
| 김주빈 | 팀장 | 회의 진행, Player 구현                             |
| 손영준 | 팀원 | Github 관리 및 머지 담당, 장애물 생성 및 파괴 담당 |
| 경원철 | 팀원 | 아이템 스폰 및 아이템 효과 구현                    |
| 장태현 | 팀원 | 화면 연결 및 UI구현                                |
| 양승민 | 팀원 | 게임 매니저 구현 및 아이템 이펙트 구현             |
